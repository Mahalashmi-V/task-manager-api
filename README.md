# Task Manager CRUD API

## Features
- Create Task
- Get All Tasks
- Get Single Task
- Update Task
- Delete Task

## Technologies
- Node.js
- Express.js
- MongoDB
- Mongoose

## Installation

npm install

## Environment Variables

Create a .env file:

PORT=5000
MONGODB_URI=your_mongodb_connection_string

## Run Project

npm start

## API Endpoints

POST /api/tasks
GET /api/tasks
GET /api/tasks/:id
PUT /api/tasks/:id
DELETE /api/tasks/:id
